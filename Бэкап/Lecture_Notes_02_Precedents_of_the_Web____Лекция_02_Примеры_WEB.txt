Классификация гипертекстовых систем (Frank Halasz)

Френк Халас из MCC дал последнюю лекцию на практикуме. Нужно попенять его и организационный комитет за то, что доклад не был первым И последним: часть лекции была хорошим обзором на то, чем по настоящему является ГТ и классификация текущих систем. Этот материал мог заполнить собой без проблем целую лекцию, но был презентован с такой скоростью, что у аудитории перехватило дыхание. Могла возникнуть отличная платформа для дискуссий в течении конференции если бы она была презентована в самом начале, нежели в конце.

Определение Гипертекста

Системы ГТ могут быть поделены с одной стороны на "оригинальное" поколение Мемекса [Ванавар Буш], NLS/Augment [Энгельбарт], Xanadu [Тэд Нельсон], и т.д. и с другой стороны "текущее" поколение состоящее из:
- Исследовательские системы: Intermedia [Brown University], NoteCards [Xerox]
- Продукты ПК: Guide [Owl], Hypercard [Apple]
- Продукты для рабочих станций: Document Examiner [Symbolics].

Ключевые компоненты изобретения Тима Бернерс Ли

URI (Uniform Resource Identifier - унифицированный идентификатор ресурса)
- глобальные названия для ресурсов

HTML
- впервые изобретено Бернерсом Ли в 1989
- основано на SGML (Standard Generalized Markup Language — стандартный обобщённый язык разметки) (использовалось для документов в ЦЕРН)
- якорный элемент: ассоциативная связь с текстом 

HTTP
- первая версия (0.9, 1991) имеет только метод GET
- запрос: получить файл по указанному пути; ответ: возврат файла

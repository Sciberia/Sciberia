4. Use f''(x) to find out whether the graph is concave up or concave down.
f'(x) = (lnx)-1 - (lnx)-2
So
f''(x) = -(lnx)-2 1/x - 2(lnx)-3 1/x
= -(lnx)-2 + 2(lnx)-3/x (lnx)3/(lnx)3 = -lnx + 2/x(lnx)3
f''(x) = 2 - lnx/x(lnx)3

#39

4. Используем f''(x) для того, чтобы определить, где график вогнут вверх или вниз.
f'(x) = (ln(x))^(-1) - (ln(x))^(-2)
Так имеем
f''(x) = (1/x)*(-ln(x)^2) - (1/x)*(-2*ln(x)^3) = (-(ln(x))^(-2)+2(ln(x))^(-3))/x * (ln(x))^3/(ln(x))^3 = (-ln(x)+2)(x*ln(x)^3)
f''(x) = (2 - ln(x))/(x*ln(x)^3)

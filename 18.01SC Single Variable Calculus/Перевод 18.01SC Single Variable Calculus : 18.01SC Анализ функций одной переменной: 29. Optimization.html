Maximum Area of Two Squares

#9

Максимальная площадь двух квадратов

Keyten 18.03.17 в 13:25

0

Consider a wire of length 1, cut into two pieces. Bend each piece into a square.

#10

We want to figure out where to cut the wire in order to enclose as much area in
the two squares as possible.

#11

In all of these problems you start with a “bunch of words” — a story problem.

#12

The two main tasks in starting the problem are to draw a diagram and to pick
variables.

#13

Figure 1: Two pieces of wire enclose two squares.

#14

If we cut the wire so that one piece has length x, the other piece will have
length 1 − x. We know that we’re bending these pieces of wire into squares, so
we can add those squares to our diagram. The first square will have sides of
length x and the second square will have sides of length 1-x/4.

#15

We want to find a maximum area, so we’ll need formulas for the areas of these
squares. The first square’s area is x2/16 and the second square has area (1-x/4)2.

#16

The total area is then
A =(x/4)2 + (1 −x/4)x2 = x2/16 + (1 − x)2/16.

#17

Most calculus students’ instinct at this point is to find the critical points —
the values x0 for which A0(x0) = 0.

#18

We can do that now if you like:
A' = 2x/16 + 2(1-x)/16 (-1)
= x/8 - 1/x + x/8
A' = 2x-1/8
A0 = 0 =⇒ 2x − 1 = 0 =⇒ x = 1/2

#19

So x =1/2 is a critical point with critical value:
A(1/2) = (1/2/4)2 + (1/2/4)2 = 1/32

#20

We’re not done yet, though. We still need to check the endpoints! The
length x of the first piece of wire has to satisfy 0 < x < 1, so we should check
the limits as x approaches the endpoints.

#21

In this case we can find those limits just by plugging in values. At x = 0,
A(0+) = 02 + (1/0/4)2 = 1/16
At x = 1,
A(1-) = (1/4)2 + 02 = 1/16

#22

If we try to graph the function with the information we have now, we see
that it starts at the point (0, 1/16), dips down to the point (1/2,1/32), then goes
back up to (1,1/16). (See Fig. ??.)

#23

Figure 2: Graph of the area function.

#24

When we found the critical point we did not find the maximum enclosed area
— the minimum area was achieved at x =1/2. The maximum area is not achieved in 0 < x < 1, but it is achieved at x = 0 or 1. The maximum corresponds to using the whole length of wire for one square.

#25

Moral: If you don’t pay attention to what the function looks like you may
find the worst answer, rather than the best one.

#26

We conclude that the least area enclosed by the two squares is 1/32, when x = 1/2; i.e when the two squares are equal. The greatest area enclosed is 1/16 when x = 0 or x = 1 and there is only one square.

#27

Commonly asked questions:
What is the minimum? It’s the minimum value 1/32.
Where is the minimum? It’s at the critical point x =1/2.

#28

These are two different questions. Be sure to answer the correct one; you
may get so involved in doing the calculus to find 1/2 that you forget to find the minimum value 1/32.

#29

Both the critical point and the critical value are important; together they form the point on the graph (1/2,1/32) where it turns around.

#30

There are many more and less precise ways to ask these questions. You’ll
have to do your best to understand what the questions (and answers) mean
from the context of the question or example.

#31

Question: Since the goal was to enclose as much area as possible, why did
we find the minimum area?

#32

Answer: The reason is that when we go about our procedure of looking
for the least or the most, we’ll automatically find both. We won’t know which
one is which until we compare values.

#33

It’s actually to your advantage to figure
out both the maximum and minimum whenever you answer such a question;
otherwise you won’t understand the behavior of the function very well.

#34

Question: Could we also use the second derivative test here?

#35

Answer: Yes, and we’re going to see an example of the second derivative
test soon. We could also look at the equation of A(x) and notice that the graph
must be a parabola that opens upward.

Introduction to Antiderivatives

#1

This is a new notation and also a new concept. $G(x) = \int g(x)dx$ is the antiderivative of $g$.

#2

Other ways of saying this are:
$$G'(x) = g(x)\quad \textrm{or,} \quad dG = g(x) dx$$

#3

There are a few things to notice about this definition. It includes a differential $dx$. It also includes the symbol $\int$, called an integral sign; the expression $\int g(x) dx$ is an integral.

#4

Another name for the antiderivative of $g$ is the indefinite integral of $g$. (We’ll learn what “indefinite” means in this context very shortly.)

#5

If $G(x)$ is the antiderivative of $g(x)$ then $G'(x) = g(x)$. To find the antiderivative of a function $g$ (to integrate $g$), we need to find a function whose derivative is $g$.

#6

In practice, finding antiderivatives is not as easy as finding derivatives, but we want to be able to integrate as many things as possible. We’ll start with some examples.

#7

Example: $\sin x$

#8

We start with the integral of $g(x) = \sin x$. This is a function whose derivative is $\sin x$. What function has $\sin x$ as its derivative?

#9

Student: $- \cos x$

#10

Because the derivative of $-\cos x$ is $\sin x$, this is an antiderivative of $\sin x$.

#11

$\textrm{If}\colon$
\begin{align}
G(x) &= -\cos x,\quad \textrm{then} \\
G'(x) &= \sin x
\end{align}

#12

$\textrm{Если}\colon$
\begin{align}
G(x) &= -\cos x,\quad \textrm{то} \\
G'(x) &= \sin x
\end{align}

#13

Because the derivative of a constant is $0$, we can add any constant to $G(x)$ and still have an antiderivative of $\sin x$.

#14

We write:
$$\int \sin x \,dx = -\cos x + c$$
and call this the indefinite integral of $\sin x$ because $c$ can be any constant — it’s an indefinite value. Whenever we take the antiderivative of something our answer is ambiguous up to a constant.

#15

Antiderivative of $x^a$

#16

What function has the derivative $x^a$? We know that the exponent decreases by one when we differentiate, so we guess $x^{a+1}$.

#17

This doesn’t quite work:
$$d(x^{a+1}) = (a + 1) x^{a} dx.$$

#18

Не похоже, что это работает:
$$d(x^{a+1}) = (a + 1) x^{a} dx.$$

\begin{align}
d(\frac{x^{a+1}}{a+1}) &= x^a dx \\

\begin{align}

\end{align}

But wait! Although it’s true that $d(x^{a+1}) = (a+1) x^a dx$, it is not always true that $\int x^a dx = \frac{x^{a+1}}{a+1} + c$.

#20

When $a = −1$ the denominator is zero. However, we can still say that $\int x^a dx = \frac{x^{a+1}}{a+1} + c$ for $a \neq -1$.

#21

What happens when $a = −1$? What is $\int \frac{1}{x} dx$?

#22

So far we’ve used the formulas $\frac{d}{dx} \cos x = -\sin x$ and $\frac{d}{dx} x^{n+1} = (n + 1) x^n$. An important part of integration is remembering formulas for derivatives and “reading them backward”.

#23

In this case, the formula we need is $\frac{d}{dx} \ln x = \frac{1}{x}$. Using this, we get $\int \frac{1}{x} dx = \ln x + c$.

#24

This formula is fine when $x > 0$, but $\ln x$ is not defined when $x$ is negative. The more standard form of this equation is:
$$\int \frac{1}{x} dx = \ln |x| + c.$$

#25

The absolute value doesn’t change anything when $x \ge 0$, so we only need to check this formula when $x$ is negative.

#26

In order to do so, we have to differentiate $\ln |x|$.
\begin{align}
\frac{d}{dx} \ln |x| &= \frac{d}{dx} \ln (-x) \quad \textrm{($|x| = -x$ when $x < 0$)} \\
&= \frac{1}{-x} \frac{d}{dx} (-x) \quad \textrm{(by the chain rule)} \\
&= - \frac{1}{-x} \\
&= \frac{1}{x}
\end{align}

#27

If we graph $\ln |x|$ we can see that this function does have slope $\frac{1}{x}$.

#29

Antiderivatives of $\sec^2 x$ and $\frac{1}{\sqrt{1 - x^2}}$

#30

Example: $\int \sec^2 x\, dx$

#31

Searching for antiderivatives will help you remember the specific formulas for derivatives. In this case, you need to remember that $\frac{d}{dx} \tan x = \sec^2 x$.
$$\int \sec^2 x\,dx = \tan x + c$$

#32

Example: $\frac{1}{\sqrt{1 - x^2}}$

#33

An alternate way to write this integral is $\int \frac{dx}{\sqrt{1 - x^2}}$. This is consistent with the idea that $dx$ is an infinitesimal quantity which can be treated like any other number.

#34

We remember $\frac{d}{dx} \sin^{−1} x = \frac{1}{\sqrt{1 - x^2}}$ and conclude that:
$$\int \frac{dx}{\sqrt{1 - x^2}} = \sin^{−1} x + c.$$

#35

Example: $\frac{1}{1+x^2}$

#36

$$\int \frac{dx}{1+x^2} = \tan^{-1} x + c$$

#37

When looking for antiderivatives, you’ll spend a lot of time thinking about derivatives. For a little while you may get the two mixed up and differentiate where you were meant to integrate, or vice-versa. With practice, this problem goes away.

Antiderivatives are Unique up to a Constant

#46

Theorem: If $F'(x) = f(x)$ and $G'(x) = f(x)$, then $F(x) = G(x) + c$.

#47

In other words, once we’ve found one antiderivative of a function we know that any other antidervative we might find will only differ from it by some added constant.

#48

Proof: If $F' = G'$ then $(F − G)' = F' − G' = f − f = 0$

#49

Recall that we proved as a corollary of the Mean Value Theorem that if a function’s derivative is zero then it is constant. Hence $G(x) − F(x) = c$ (for some constant $c$). That is, $G(x) = F(x) + c$.
